<h2>第3章 Koa2基础知识</h2>
<h3>安装koa-generator:npm install -g koa-generator</h3>
<h3>初始化项目:koa2 -e koa</h3>
<p>-e是指定ejs模板，默认jade模板，koa为项目名称</p>
<p>进入到koa目录，npm install & npm start (开发模式:npm run dev)启动项目；访问项目localhost:3000</p>
